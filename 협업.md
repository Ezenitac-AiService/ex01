### 개인용 github 계정의 저장소에 협업자 등록으로 협업하기

+ 저장소의 settings 메뉴
  + Collaborators 메뉴
    + Manage access
    + add people
      + username, fullname, email 등으로 협업자 초대 보내기
      + 초대 받은 계정은 초대 수락하기

+ 해당 github 저장소는 협업자 모두에게 clone, commit, push 권한이 주어짐
  + 저장소를 컴퓨터에 내려받기
    + git clone 저장소주소
  + 파일 작업 후 add, commit, push로 github 저장소에 작업 이력을 등록할 수 있습니다

+ **clone시 github 저장소의 main 브랜치를 가져옵니다**
  + main은 정말 조심히 관리해야 합니다
  + main 브랜치를 복제한 가지치기 브랜치를 이용해야 합니다

+ 브랜치 생성
```bash
git branch 이름
```
+ 브랜치 목록 확인
```bash
git branch
```
+ 브랜치 변경
```bash
git switch 브랜치이름
```

+ 브랜치 push하기
```bash
git push -u origin 브랜치이름
```
